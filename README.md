# AG-Mapping
An application for 3D mapping tasks in agriculture uses object-based Visual SLAM 

# Requirements

- Ubuntu 20.04
- Qt 5.15.2
- PCL 1.10
- OpenCV 4.2
- Pangolin
- Boost
- CUDA 11.6
- ZED SDK (Option)
- Realsense SDK (Option)
